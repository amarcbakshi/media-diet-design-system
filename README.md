# Media Diet — Design System

Brand design system for [Media Diet](https://ourmediadiet.com), an immersive documentary installation by Amar C. Bakshi and Heidi Boisvert.

## For Claude Design

Point Claude Design at this repository during setup. Claude will read `DESIGN.md` as the primary source of truth for all design decisions.

**What's included:**
- `DESIGN.md` — complete design language (colors, type, logos, components, photography, voice)
- `logos/` — SVG logo files, both Normal and Glass variants
- `images/glass/` — brand photography with glass-effect treatment (26 images)
- `tokens/` — CSS custom properties for color, typography, spacing
- `components/` — CSS component patterns

**Font files:** Suisse Int'l Bold and Medium are required but not included here (licensed). Add `SuisseIntl-Bold.otf` and `SuisseIntl-Medium.otf` to a `/fonts/` directory. Available from Swiss Typefaces.

## Quick Summary

Three colors: `#000000` (black), `#FFFFFF` (white), `#00FFFF` (cyan — accent only, on dark/photo backgrounds exclusively).  
Typeface: Suisse Int'l Bold (headlines) + Medium (body).  
Logo: Normal logo over glass-effect imagery; Glass logo over normal photography.  
Principle: restraint amplifies confrontation. The UI disappears so the content lands.
